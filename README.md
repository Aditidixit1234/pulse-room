# PulseRoom – Real-Time Group Study & Collaboration Platform

PulseRoom is a real-time collaboration platform that enables students and professionals to create study rooms, share resources, communicate through chat, and collaborate efficiently.

## Features

* Room creation and management
* Real-time messaging using Socket.IO
* User authentication and authorization
* Room participant management
* Collaborative study sessions
* Resource sharing
* Online user tracking
* Notification system
* Public and private rooms

## Tech Stack

### Frontend

* Next.js
* TypeScript
* Tailwind CSS

### Backend

* FastAPI
* Python Socket.IO

### Database

* PostgreSQL

### Authentication

* JWT Authentication

## Architecture

* FastAPI REST APIs
* WebSocket communication
* PostgreSQL persistent storage
* Scalable modular backend
